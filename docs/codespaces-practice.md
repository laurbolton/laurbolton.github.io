## Understanding Git Workflow
Explain in 2–3 sentences the difference between:
- Local workspace (Codespace)
- Remote repository (GitHub.com)  
In a local workspace, the user makes changes to a clone of the remote repository that exists in a virtual environment. Changes made on Codespace will not automatically save to the remote repository on GitHub. Any modifications need to be offloaded (staged, committed, and pushed) from the local workspace via the source control panel on Codespace.
